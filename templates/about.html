Project Overview

Eye Disease Detection is an academic capstone project that applies transfer learning and ensemble deep learning to automate the classification of eye diseases from fundus (retinal) photographs. The system was trained on the Eye Diseases Classification dataset sourced from Kaggle, comprising approximately 4,217 labeled retinal images across four categories.

The core methodology follows a multi-model ensemble approach: eight pre-trained convolutional neural network (CNN) architectures are fine-tuned independently using transfer learning from ImageNet weights. Predictions from individual models are aggregated via probability averaging to produce a final, robust classification.

A CLAHE (Contrast Limited Adaptive Histogram Equalisation) preprocessing step is applied on the L-channel of the LAB colour space to enhance retinal image contrast before inference, mirroring the training pipeline exactly.

Ensemble Strategy

Each model produces a 4-dimensional probability vector via softmax. Predictions from all enabled models are averaged element-wise:

Pfinal(c) = (1/N) × Σ Pi(c)

The class with the highest averaged probability is selected as the final prediction. If ensemble confidence falls below a configurable threshold (default 65%), the system automatically consults external API fallbacks for re-validation:

  1. Hugging Face Inference API (free tier)
  2. Google Gemini Vision API (free tier)

Final result is a 60/40 weighted blend of ensemble and API outputs.
